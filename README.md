A complete authentication + task management system built with **Laravel 12**, **Vue.js 3 (Inertia)**, and **MySQL**.

## Features

### Authentication
- User Signup (username, email, password)
- Login with Email + Password
- Login / Signup with **Login With Gmail**
- Forgot Password
- Reset Password

### Profile Management
Users can update:
- Name  
- Profile Photo  
- Date of Birth  
- Education  

### Task Management (After Login)
- Add Task  
- List Tasks  
- Update Task  
- Delete Task

## Requirements

Make sure the following are installed:

- PHP 8.1
- Composer
- Node.js 18

- Vue js

- XAMPP Server

---

## Project Installation

### Run these commands to install the project 

- composer install
- npm install
- Create .env file from .env.example
- php artisan key:generate
- Create database in mysql named:  taskmate
- php artisan migrate
- npm run dev
- npm run build
- php artisan serve    (To run the project)

### Authorized URL 
http://127.0.0.1:8000/
